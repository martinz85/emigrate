## Die 10 kritischen Schritte

### Schritt 1: Zielland festlegen (Woche 1-2)

**Schnell-Check für dein Zielland:**

1. EU-Bürger? → EU-Länder sind am einfachsten
2. Remote Worker? → Digital Nomad Visa prüfen
3. Budget knapp? → Portugal, Spanien, Griechenland
4. Englisch reicht? → Malta, Zypern, Niederlande, Skandinavien
5. Sonne wichtig? → Südeuropa, Kanaren

**Entscheidungsbaum:**

```
Bist du EU-Bürger?
├── JA → Jedes EU-Land möglich
│   └── Tipp: Portugal, Spanien, Griechenland (günstig + warm)
└── NEIN → Visa-Optionen prüfen
    ├── Hast du Remote-Einkommen?
    │   └── JA → Digital Nomad Visa (Portugal D7, Spanien, Griechenland)
    └── NEIN → Arbeitsvisum oder Investorenvisum nötig
```

### Schritt 2: Finanzen checken (Woche 1)

**Minimum-Puffer:** 6 Monate Lebenshaltungskosten + Umzugskosten

**Schnelle Kalkulation:**

| Posten | Einmalig | Monatlich |
|--------|----------|-----------|
| Umzug (Container) | 3.000-8.000 EUR | - |
| Flüge | 200-1.000 EUR | - |
| Kaution Wohnung | 2-3 Monatsmieten | - |
| Lebenshaltung | - | 1.500-4.000 EUR |
| Krankenversicherung | - | 200-500 EUR |

**Faustregel:** Plane mit **15.000-25.000 EUR** für einen soliden Start.

### Schritt 3: Visa/Aufenthalt klären (Woche 2-4)

**Für EU-Bürger:**
- Keine Genehmigung nötig für Aufenthalt
- Nach 3 Monaten: Anmeldung beim Rathaus
- Fertig.

**Für Nicht-EU-Bürger:**
- Visa-Typ identifizieren
- Dokumente sammeln (kann Wochen dauern!)
- Antrag bei Botschaft/Konsulat

### Schritt 4: Wohnung finden (Woche 4-8)

**Strategie für Schnelle:**

1. **Airbnb für 1-2 Monate** - Flexibel ankommen
2. **Vor Ort suchen** - Nicht remote mieten
3. **Lokale Portale nutzen** - Nicht nur Immoscout

**Portale nach Land:**
- Portugal: Idealista, OLX
- Spanien: Idealista, Fotocasa
- Zypern: Bazaraki
- Griechenland: Spitogatos

### Schritt 5: Abmeldung Deutschland (Woche 6-8)

**Checkliste:**

- [ ] Einwohnermeldeamt: Abmeldung
- [ ] Finanzamt: Wegzug melden
- [ ] Krankenversicherung: Kündigen/Anwartschaft
- [ ] Rentenversicherung: Informieren
- [ ] Bank: Adresse ändern (oder kündigen)
- [ ] Post: Nachsendeauftrag

### Schritt 6: Anmeldung neues Land (Woche 8-10)

**Typische Schritte:**

1. Rathaus/Gemeinde → Anmeldung
2. Steuernummer beantragen
3. Sozialversicherungsnummer (falls nötig)
4. Bankkonto eröffnen

### Schritt 7: Bankkonto (Woche 8-10)

**Optionen:**

- **Lokale Bank** - Oft langsam, viel Papierkram
- **N26/Revolut** - Schnell, EU-weit nutzbar
- **Wise** - Für Multi-Währungen ideal

### Schritt 8: Krankenversicherung (vor Umzug!)

**Optionen für EU-Bürger:**

- EHIC-Karte (temporär, erste Monate)
- Lokales System (nach Anmeldung)
- Private Auslandskrankenversicherung

**Empfehlung:** Erst private Versicherung, dann lokales System.

### Schritt 9: Alltag organisieren (Monat 2-3)

- [ ] Handyvertrag (lokale SIM)
- [ ] Internet-Anschluss
- [ ] Strom/Gas (falls nicht inklusive)
- [ ] Führerschein (EU-weit gültig)
- [ ] Auto ummelden (nach 6 Monaten Pflicht)

### Schritt 10: Netzwerk aufbauen (laufend)

**Schnelle Wins:**

- Facebook-Gruppen: "[Stadt] Expats", "Deutsche in [Land]"
- Meetup.com: Events für Neuankömmlinge
- InterNations: Globale Expat-Plattform
- Sprachkurse: Doppelter Nutzen

## Die 5 größten Fehler

1. **Zu wenig Puffer** → Mindestens 6 Monate
2. **Wohnung remote mieten** → Immer vor Ort besichtigen
3. **Sprache unterschätzen** → Basics vor dem Umzug lernen
4. **Steuern ignorieren** → Kann teuer werden
5. **Allein planen** → Partner/Familie einbeziehen
